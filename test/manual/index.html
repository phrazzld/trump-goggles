Trump Goggles - Manual Test Suite

Note: This test suite is designed to verify the functionality of the Trump Goggles browser extension. Make sure the extension is installed and active before running these tests.

Static Content Test

Tests how the extension handles various static text content with references to Donald Trump and related terms.

Run Static Content Test

Test Checklist:

  • All variations of "Donald Trump" should be replaced
  • Standalone "Trump" references should be replaced
  • Text in form fields should NOT be modified
  • Text with HTML entities should be correctly processed
  • Nicknamed references should be handled if configured

Dynamic Content Test

Tests how the extension handles content that is dynamically added to the page after initial load.

Run Dynamic Content Test

Test Checklist:

  • Content added via DOM manipulation should be processed
  • Content loaded via AJAX should be processed
  • Content that updates at intervals should be processed
  • Content added by dynamically inserted scripts should be processed

Error Simulation Test

Tests how the extension handles various error conditions and edge cases.

Run Error Simulation Test

Test Checklist:

  • Extension should not crash with invalid DOM structures
  • Extension should handle deeply nested DOM elements
  • Extension should not cause infinite loops with circular references
  • Extension should process large numbers of text nodes without significant lag
  • Extension should handle rapid mutations without crashes
  • Extension should not enter recursive mutation loops
  • Extension should properly process delayed content
  • Extension should handle race conditions gracefully

Browser Compatibility

Run all tests in different browsers to verify cross-browser compatibility.

Browsers to Test:

  • Chrome (latest stable version)
  • Firefox (latest stable version)
  • Edge (if available)
